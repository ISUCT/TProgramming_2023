Игра
	Каждый герой за свой ход может либо совершить обычную атаку, либо применить одну из своих способностей.
	
	У каждого героя урон представлен в виде диапазона [<минимальный>, <максимальный>]. При совершении атаки урон будет принимать любое вещественное значение из этого диапазона.
	
	При совершении обычной атаки у каждого героя есть шанс промахнуться.


Герои

	Fighter
		Способности
			Smash - совершает атаку с критическим уроном 215% от текущего урона.
			Невозможно промахнуться.
			Шанс срабатывания: 19.5%.
	
	Archer
		Способности
			Arson - поджигает цель и заставляет её каждый свой ход получать 44% от среднего урона Archer.
			Невозможно промахнуться.
			Наносит урон даже если цель промахивается атакой.
			Возможно единственное применение за раунд.
			Шанс срабатывания: 30%.
	
	Mage
		Способности
			Oblivious - заставляет цель пропустить 2 хода.
			Невозможно применить два раза подряд.
			На следующем ходу после применения способности Mage имеет повышенный шанс промахнуться базовой атакой (18.5%).
			Шанс срабатывания: 25%.



Запуск
	node ./path/to/game.js <кол-во героев>
